# southkorea_election
South Korean election forensics

The census data (csv files) are from https://www.data.go.kr/data/3033304/fileData.do (official Korean government census data), which directs to https://jumin.mois.go.kr/ageStatMonth.do .

The elction results (csv files) are from official Korean government data, https://www.data.go.kr/data/15025527/fileData.do

The actual monthly housing price data are from official Korean government data, https://www.data.go.kr/data/3050988/fileData.do

The previous voting data sqlite is given in https://github.com/freedomfighter2022/koreaelection2020, with the raw data provided by the Korean NEC. I now changed the code to use the official csv data instead.
